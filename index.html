Framework

How it Works

01 Content ID

Assigns a persistent unique ID to each creative asset for tracking across all channels.

02 Exposure

Tracks exposure using pixel/server-to-server to capture device IDs.

03 Identity Match

Matches the exposure data to individual user identities deterministically.

04 Attribution

Proprietary ML assigns outcomes to the specific content that drove the actions.

05 Outcome

Measures outcomes such as site visits, lift, and conversions.

06 Optimization

ML scores performance to provide media buy recommendations.
