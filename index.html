Framework

How it Works

01Content ID

Assigns a persistent ID unique identifier to each creative asset, enabling tracking and analysis across channels.

02Exposure

Tracks the exposure of content using a pixel/server-to-server/clean room to capture the device IDs.

03Identity Match

Matches the exposure data to individual user identities.

04Attribution

Proprietary ML assigns attribution of outcomes to the specific content that drove the desired actions.

05Outcome

Measures outcomes such as site visits, lift, and conversions associated with the content.

06Optimization

Proprietary ML scores and ranks content performance, providing media buy recommendations.
