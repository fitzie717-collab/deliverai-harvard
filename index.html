The Framework

The Technical Workflow

01Content Identification

Assigns a persistent unique identification to each creative asset to enable cross-channel tracking.

02Exposure

Tracks content exposure using pixel/server-to-server or clean room to capture device identifications.

03Identity Match

Matches the exposure data to individual user identities deterministically.

04Attribution

Proprietary Machine Learning assigns outcomes to the specific content that drove actions.

05Outcome

Measures outcomes such as site visits, brand lift, and conversions associated with content.

06Optimization

Machine Learning scores content performance to provide media buy recommendations.
